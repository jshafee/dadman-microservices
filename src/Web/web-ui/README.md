# Web UI Workspace (Curated)

This directory is intentionally kept minimal.

- Full imported Velzon template source is stored in `../web-ui-template`.
- Move only approved/needed sections from `web-ui-template` into this folder based on product requirements.

Suggested workflow:
1. Copy a required page/component/layout from `../web-ui-template/src`.
2. Adapt it to project routing/state conventions.
3. Remove unused dependencies from `package.json` once the curated UI stabilizes.
